# 3.实现逻辑

  • col 使用margin:0 gutter px
  • row 使用margin:0 -gutter/2 px
  • 同时在row使用 flex-wrap:nowrap
  • offset 使用了 margin-left 来设置
  • 跨度样式使用了SASS的forEach
$class-prefix: col-;
@for $n from 1 through 24 {
    &.#{$class-prefix}#{$n} {
        width: ($n / 24) * 100%;
    }
}
